UBUNTU ARM STEAMCMD SERVER

By default, most game servers do not support ARM CPU's, which is the only available config for Oracles Free tier with decent specs. To install steam CMD, we need Box-86 64. Your server will give you errors, but ignore them, as it works regardless. This guide does not cover port fowarding or getting the oracle cloud instance in the first place. 




Getting Started


 sudo apt update 
sudo apt full-upgrade



Installing Box-86


sudo apt install git build-essential cmake
git clone https://github.com/ptitSeb/box86
sudo dpkg --add-architecture armhf
sudo apt update
sudo apt install gcc-arm-linux-gnueabihf
sudo apt install libc6:armhf
sudo apt install libncurses5:armhf
sudo apt install libstdc++6:armhf
cd box86
mkdir build
cd build
cmake .. -DRPI4ARM64=1 -DCMAKE_BUILD_TYPE=RelWithDebInfo
make -j$(nproc)
sudo make install
sudo systemctl restart systemd-binfmt


Installing Box-64



cd ../..
git clone https://github.com/ptitSeb/box64.git
cd box64
mkdir build
cd build
cmake .. -DRPI4ARM64=1 -DCMAKE_BUILD_TYPE=RelWithDebInfo
make -j$(nproc)
sudo make install
sudo systemctl restart systemd-binfmt



Installing Steam CMD



cd ../..
mkdir steamcmd
cd steamcmd
curl -sqL "https://steamcdn-a.akamaihd.net/client/installer/steamcmd_linux.tar.gz" | tar zxvf -




Running Steam CMD


./steamcmd.sh

 if it runs, wait until it installs then type quit. 





 Downloading your Garry's Mod server 


./steamcmd.sh +@sSteamCmdForcePlatformType linux +force_install_dir /home/ubuntu/kinfogaming +login anonymous +app_update 4020 validate +quit




 Starting your Garry's Mod server 


cd kinfogaming/
screen -A -m -d -S gmod ./srcds_run -console -game garrysmod +maxplayers 16 +map rp_downtown_v4g_v2 +gamemode darkrp +host_workshop_collection WORKSHOPID +sv_setsteamaccount STEAMACCOUNTKEY 










	
	
For safety & security, you should make a new user for steamcmd & your server, although not nessesary
